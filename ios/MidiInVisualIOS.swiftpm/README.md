# MidiInVisualIOS

Small native iOS MIDI input visualizer.

Purpose:

- prove that an iPad app can receive MIDI from the Roland FP-90X
- keep the test independent from tabla/Ruscelli
- visualize incoming 88-key note on/off data like `midi_in_visual_01.html`

Open `MidiInVisualIOS.swiftpm` in Xcode, select an iPad target, and run.

Use the `Bluetooth MIDI` button to open Apple's Bluetooth MIDI device browser.
